**Instructions for genomic analysis**

*Note (4-15-26) - this page is still in progress. Do not follow tutorials unless CS told you they are done, I am using this mostly as a placeholder for notes!*

Genomics is the analysis of single genomes (rather than metagenomics, which looks at all of the genetic material in a sample). Genomic data is acquired the same way as metagenomic (DNA extraction and sequencing), but is almost always on pure cultures so that the sequence data all belongs to the same organism. It is analyzed in different ways than metagenomic data - metagenomics often focuses on identifying all of the species in an environment, whereas genomics looks for changes in an individual genome compared to a reference, or aligns many genomes from the same organism to look at how they are related.
